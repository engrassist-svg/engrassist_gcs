This Excel template will include the following columns for comprehensive VAV terminal unit documentation:

Basic Information

  • Mark Number: Equipment tag (e.g., VAV-1, VAV-2)
  • Zone Name: Space served
  • Zone Type: Interior, Perimeter, Conference, etc.
  • Floor Level: Floor location

Airflow Requirements

  • Maximum CFM: Peak cooling airflow
  • Minimum CFM: Minimum ventilation airflow
  • Turndown Ratio: Max/Min ratio
  • Supply Air Temperature: Typically 55¬∞F

Equipment Specifications

  • Type: Cooling-only, HW Reheat, Electric Reheat, Fan-Powered, etc.
  • Control Type: Pressure Independent (DDC) or Pressure Dependent
  • Inlet Size: Round or rectangular duct size
  • Sound Rating: NC rating requirement
  • Acoustic Liner: Yes/No

Reheat Information (if applicable)

  • Reheat Type: Hot Water, Electric, None
  • Heating Capacity: BTU/hr or kW
  • Coil Rows: 1-row, 2-row (for HW)
  • Hot Water Temperature: 180¬∞F typical
  • Electric Voltage: 120V, 208V, 240V (for electric reheat)

Controls & Integration

  • Controller Model: Manufacturer's controller part number
  • Communication Protocol: BACnet, Modbus, LonWorks
  • Thermostat Type: Room sensor model
  • Actuator Voltage: 24VAC typical

Additional Options

  • Airflow Sensor Type: Hot wire, Differential pressure
  • Damper Type: Single blade, Opposed blade
  • Fan Size (FP boxes): HP rating for fan-powered units
  • Filter (FP boxes): MERV rating

Installation Notes

  • Location: Ceiling location/grid reference
  • Access Panel: Required ceiling access panel size
  • Manufacturer: Approved manufacturers (Titus, Price, Nailor, etc.)
  • Notes: Special requirements or considerations
